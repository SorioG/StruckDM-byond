Skin Tutorial

This tutorial will show you how to make skins for StruckDM

First, you need to use the stickman as a template, download here


1. Make a icon (.dmi file) in Dream Maker, Try making a environment for it if you don't have one.

2. Copy everything or make states in the template for your icon.

3. This step is optional, If you copy everything from the template, you need to edit them yourself

4. Once you're done in making your own skin, You need to call the command "Set Skin", click or type on it

5. Choose a icon you made and you're done

Skin Rules

1. Don't make inappropiate/NSFW skins

2. Don't make invisible skins, Your players won't see you at all, and that is cheating. (except bots because they still see you)

3. You are allowed to share your skins to others.

4. Make sure that your skin has all of the states StruckDM uses, because otherwise that will be confusing for things that use these states.

Testing your Skin

If you wish to start a test of your skin, You can now do it yourself by calling "Test Skin", but it's recommended that you test it on a private server.

StruckDM will start looking for unnecessary states, and expecting states (states that the game requires).

If all of them failed, you need to fix them yourself.
